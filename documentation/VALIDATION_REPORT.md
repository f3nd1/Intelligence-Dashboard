## Corrected regressions

1. The Source Mapping Report overlay and the visual hover menu are appended
   inside `#uccIntelligencePlatform`. This keeps them within the Custom HTML
   Block CSS scope and prevents the unformatted report from appearing below the
   dashboard.
2. Dynamic Source Mapping Report buttons use platform-level delegated click
   handling. The hero View tools action continues to use the shared custom
   event.
3. The Explore and hover catalogues use approved visual definitions only, not
   hidden legacy chart containers.
4. Grouped panels create separate visual grids for each exact subcriterion and
   hide inactive grids.
5. The blank-visual guard waits 20 seconds and skips dashboards with an active
   loading overlay.
6. Source diagnostics return only source groups approved for the selected
   criterion, which removes duplicate shared rows and false Provider Rating
   errors.
