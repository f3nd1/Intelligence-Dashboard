The policy defines the ARTEMIA closed-loop approach:

1. Assessment
2. Review
3. Targeting
4. Execution and Mitigation
5. Improvement
6. Accountability

Performance outcomes are grouped into student and graduate, service quality,
operational and people-development outcomes. The original policy file is
preserved under `source-material/`.
