# Console Error Assessment — v1.8.7

## Platform defect addressed

```text
Error: <path> attribute d: Expected number, "MNaN,NaNLNaN,NaNZ".
```

The package now applies finite-number guards to the generic live charts and
Criterion 4 policy-flow diagrams. Live UAT is still required because other site
scripts may also create SVG charts.

## Errors outside the Custom HTML Block

The following console messages are not generated by active UCC Intelligence
Platform source code:

### Browser extensions

```text
chrome-extension://hdokiejnpimakedhajhdlcegeplioahd/
background-redux-new.js
[VAULT]
duplicate id LastPass
message channel closed before a response was received
```

These originate from a password-manager/browser extension. The platform cannot
repair extension internals.

### Missing Educ Sg asset

```text
/assets/educ_sg/survey.min.css 404
```

No active platform HTML, CSS or JavaScript references `survey.min.css`. This
must be corrected in the Educ Sg asset bundle, hooks, template or site cache.

### Calendar custom script

```text
custom_calendar_override9.js
Cannot read properties of null (reading 'doctype')
```

This is a separate site custom script. It must guard the page/form object before
reading `doctype`.

### `228.js` object logging

Repeated `Object` messages from `228.js` are outside the platform package. The
source asset should be inspected separately if the logging is unwanted.

### Icon preload warnings

Frappe icon preload warnings are informational unless icons fail to render.
